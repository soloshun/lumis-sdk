# Lumis SDK support

Lumis SDK is pre-alpha research software maintained on a best-effort basis.

- Use GitHub issues for reproducible bugs, documentation errors, and focused feature proposals.
- Use an RFC discussion before cross-cutting architectural work.
- Do not report vulnerabilities or exposed credentials in public issues; follow `SECURITY.md`.
- Include Lumis SDK version, Python version, operating system, minimal synthetic reproduction, and relevant command output.
